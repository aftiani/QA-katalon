## 🛠 Tools & Versi
* **Katalon Studio**: v11.1.3
* **Java**: v17
* **Browser**: Google Chrome v12x (untuk Web UI Testing)
* **OS**: Windows

* `Test Cases/` :
* TC01-Login
* TC02-Navigate
* TC03_Verify
* TC04-Add
* TC05_Search
* TC06_MultiSearch
* TC08-Delete

* `Test Suites/`: Kumpulan test case yang dijalankan secara bersamaan.

* ### 1. Melalui Katalon Studio IDE (UI)
1. Buka aplikasi Katalon Studio dan buka folder project ini.
2. Masuk ke tab **Test Explorer** di sebelah kiri.
3. Buka folder `Test Suites` dan pilih "TS ALL" untuk menjalankan semua testcase
4. Klik tombol **Run** pada toolbar bagian atas (atau tekan tombol *Play*).
